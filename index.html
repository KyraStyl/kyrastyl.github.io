Selected Projects

  • LimeCEP: Hybrid Complex Event Processing Engine for Edge Environments CEP Edge Kafka
    Development of LimeCEP, a hybrid CEP engine for edge settings that combines lazy evaluation, speculative processing, and buffer-based techniques to handle out-of-order, late, and duplicate events while maintaining high accuracy and low latency on real streams with inconsistencies.
    GitHub: repository link (LimeCEP)
  • TRINEFLEX: Distributed Energy Storage Matchmaking & Networking Infrastructure Horizon Europe Energy
    Implementation of the core infrastructure for an intelligent stakeholders matchmaking and networking system, focusing on data processing, inter-service communication, and integration with BPM orchestrator (Camunda) and a Distributed Ledger (Hyperledger Fabric). Backend developed in Python (Flask REST API) with a PySpark/Delta Lake storage layer over MinIO (S3), and containerised with Docker.
  • SASEXT: Hybrid Complex Event Processing Engine (ACM SAC 2023) CEP Hybrid Automata
    Design and implementation of SASEXT, a hybrid CEP mechanism targeting efficient detection of Kleene patterns in demanding streaming environments. The system uses a lazy, hash-enhanced automata algorithm to significantly reduce memory usage and latency compared to traditional NFAs and CET engines, with support for time windows and expired event handling.
  • Edge Analytics Engine for Health Data Edge Computing HPC
    Edge system that receives patient data from various services, performs transformations and analytics, and forwards results to a cloud component. Implemented a REST API with Spring Boot, applied data transformations into a unified format, and packaged the service with Docker for ARM64 (e.g., Raspberry Pi).
  • Hybrid CEP Engine (SASE + CET) Hybrid CEP Raspberry Pi
    Unified engine that combines SASE and CET for CEP workloads, extended with new data types and sliding window support. Evaluated memory usage, CPU utilization, total execution time, and detection latency, demonstrating robust performance even in resource-constrained environments such as Raspberry Pi devices.
    GitHub: Hybrid_CEP_Engine
  • Frontend for a Cybersecurity Tool JavaScript Web UI
    Development of a JavaScript-based frontend for a cybersecurity tool, using HTML and CSS to create interactive visual interfaces for security-related functionality.
